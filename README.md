# RbxRefresh
Mirrors a local file structure to ROBLOX Studio. 

## Usage
1. Install plugin
2. Install NodeJS
3. `node YOUR_RBX_REFRESH_FOLDER YOUR_SRC_FOLDER`

Your src folder should mirror the roblox explorer structure.
i.e.
```
Project
->src
--->ReplicatedStorage
----->Classes
------->Module.lua
--->ServerScriptService
----->Server.lua
--->StarterPlayer
----->StarterPlayerScripts
------->Client.lua
```
